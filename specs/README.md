# Specs

This folder contains human-readable test plans generated by the Playwright Planner agent.

Guidance:

- Use `npm run init-agents` to initialize agent definitions (Playwright will create `.github/agents` entries).
- Create or update `tests/seed.spec.ts` to provide a seed test that bootstraps the application state.
- Run the planner agent (via the Playwright agent workflow or manually) to produce `specs/*.md` test plans.

Keep plans small and focused; the Generator agent will transform them into executable tests under `tests/`.
